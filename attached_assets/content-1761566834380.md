# WhatsApp API  Documentation

WhatsApp HTTP API for sending and receiving messages, files, media, statuses. Works with groups, communities and channels. Allows you to receive chats and messages, send products and process orders, check numbers and use many other automations.

[Developer Hub with Code Snippets](https://whapi.readme.io/reference/checkhealth)

or

Parameters in GET queries pass query string. Parameters in POST requests - through the JSON-encoded request body. Use the token as a Bearer authorization header. **[Get token for WhatsApp API!](https://panel.whapi.cloud/register)**

Examples of projects: [NodeJS bot](https://github.com/Whapi-Cloud/nodejs-whatsapp-chatbot) / [PHP bot](https://github.com/Whapi-Cloud/php-whatsapp-chatbot) / [Python bot](https://github.com/Whapi-Cloud/python-whatsapp-chatbot) / [WhatsApp Checker](https://github.com/Whapi-Cloud/whatsapp-checker).
